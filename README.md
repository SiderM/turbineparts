# Turbine Parts Catalog


Приложение поиска запчастей для ремонта турбин.



_Live: https://https://turbineparts.vercel.app_

### Настройка

```shell
$ npm install
```

### Запуск

```shell
$ npm run serve
```

### Сборка

```shell
$ npm run build
```
